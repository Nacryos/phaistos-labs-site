# Phaistos Labs v1 archive

This directory contains the complete original static site from commit `862a061` (`v1-production`). Its relative paths are unchanged, so it can still be served directly from this directory.

From the repository root, run:

```bash
npm run dev:legacy
```

Do not edit this archive as part of v2 work. Fixes intended for the old production site should be made from the `legacy/v1` branch and deliberately carried forward if needed.
